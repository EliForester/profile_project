# Profile Project
Project to create/use user profile

## Getting Started

Clone the repository

Run the following:

```
  python manage.py makemigrations
  python manage.py migrate
  python manage.py runserver 0.0.0.0:8000
```

### Prerequisites

Created on Python 3 w Django 2

requirements.txt
```
Django==2.0.3
Pillow==5.1.0
pytz==2018.3

```
